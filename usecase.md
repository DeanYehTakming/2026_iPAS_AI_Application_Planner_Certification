# 專案：2026 iPAS 認證 關鍵字典 (Use Case Specification)

## 1. 專案概述 (Project Overview)
**2026 iPAS AI 應用規劃師** 認證考試內容涵蓋廣泛，從傳統機器學習到最新的生成式 AI (GenAI) 及 AI 倫理法規。本專案旨在建立一套**結構化、情境化**的關鍵字字典，解決考生「看懂文字但不會應用」的痛點，並作為講師教學的標準化輔助工具。

## 2. 目標角色 (Stakeholders / Actors)
* **初學者/考生 (Candidates)**：需快速建立 AI 知識架構、應對情境題的人員。
* **授課講師 (Instructors)**：需標準化教學內容、快速產出課堂案例的專家。
* **專案經理 (Project Managers)**：在企業內部推動 AI 專案時，需精準溝通術語的人員。
* **內容維護者 (Editors)**：負責根據最新考綱（如 2026 版）更新定義的人員。

## 3. 核心使用案例 (Core Use Cases)

### UC-01：術語精準檢索與概念對齊
* **主要參與者**：考生、講師
* **情境**：在閱讀教材或處理 AI 專案時，遇到模糊或容易混淆的術語（例如：微調 Fine-tuning vs. 提示工程 Prompt Engineering）。
* **主要流程**：
    1. 使用者輸入關鍵字（中文或英文）。
    2. 系統展示：**標準定義**、**2026 考點摘要**及**中英對照**。
    3. 系統關聯顯示相似或相對應的詞彙（例如：檢索增強生成 RAG）。
* **預期效益**：建立清晰的知識節點，避免概念混淆。

### UC-02：2026 新考綱衝刺導讀
* **主要參與者**：考生
* **情境**：距離考試僅剩一個月，需優先掌握 2026 年新增的內容（如 EU AI Act、LLM 幻覺處理）。
* **主要流程**：
    1. 使用者選擇標籤過濾功能（例如：`#New2026`, `#GenAI`）。
    2. 系統呈現精選後的關鍵字清單。
    3. 系統根據「鑑定重點」標註該詞彙在考試中的出題型態（如：法規義務題、技術選型題）。
* **預期效益**：提高複習效率，專注於高權重考點。

### UC-03：情境化教學案例生成
* **主要參與者**：講師
* **情境**：講師在講解「資料洩漏 (Data Leakage)」或「AI 偏見」時，需要一個貼近企業實務的例子。
* **主要流程**：
    1. 講師選取特定術語。
    2. 系統展示「實務範例 (Practical Scenario)」欄位內容（如：某銀行使用 AI 審核貸款時因特徵選擇不當導致性別歧視）。
    3. 講師引用該案例與學員進行分組討論。
* **預期效益**：增加教學的互動性與實戰感，幫助學員應對 iPAS 最難的「應用情境題」。

### UC-04：自動化隨堂小測驗 (Self-Assessment)
* **主要參與者**：考生、講師
* **情境**：完成一個單元的教學後，需立即檢驗學習成效。
* **主要流程**：
    1. 系統根據字典中的「鑑定重點 (Exam Point)」自動生成單選題題目。
    2. 提供情境敘述，要求學員選出正確的對應技術或解決方案。
* **預期效益**：將「記憶」轉化為「應用」能力。

## 4. 字典資料格式 (Data Schema)

| 欄位名稱 (Field) | 類型 | 說明 | 範例 |
| :--- | :--- | :--- | :--- |
| **ID** | String | 唯一識別碼 | `AI-2026-001` |
| **Term_ZH** | String | 中文名稱 | 檢索增強生成 |
| **Term_EN** | String | 英文名稱 | Retrieval-Augmented Generation (RAG) |
| **Category** | Enum | 領域分類 | 生成式 AI / 系統架構 |
| **Definition** | Text | 核心定義 | 結合外部知識庫以提升 LLM 回答準確性的技術。 |
| **Exam_Point** | Text | 鑑定重點 | 與 Fine-tuning 的區別、減少幻覺的優點。 |
| **Scenario** | Text | 企業實務情境 | 企業內部數位客服連結產品說明手冊。 |
| **Difficulty** | Int | 難易度 (1-5) | 4 |
| **Tags** | List | 標籤 | `#GenAI`, `#New2026`, `#Architecture` |
